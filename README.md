# FUTURE_TrackCode_02
Social Media Campaign Performance Tracker:
I built an interactive Marketing Campaign Insights dashboard built using Power BI to analyze customer spending, revenue, and buying patterns. The dashboard highlights key influencers, customer segmentation by country, education, and marital status, and supports data-driven marketing and business decisions.

## Dashboard Workflow
1. Data Collection - Raw marketing and customer data were collected from structured datasets containing customer demographics, income, purchases, and campaign responses.
2. Data Cleaning & Preparation - Data was cleaned using Power Query by handling missing values, correcting data types, removing duplicates, and standardizing categorical fields.
3. Data Transformation - New calculated columns and measures were created using DAX to compute KPIs such as Total Spend, Total Revenue, Average Income, and Total Purchases.
4. Data Modeling - Relationships were established between customer, sales, and campaign tables to enable accurate filtering and cross-visual interaction.
5. Dashboard Design - Interactive visuals were created including KPI cards, bar charts, maps, pie charts, trend analysis, and key influencer visuals.
6. Filtering & Interactivity - Slicers were added for country and education level to allow dynamic analysis and user-driven exploration.
7. Insights & Validation - Results were analyzed to identify high-value segments, key spending influencers, and regional buying trends for business decision-making.
